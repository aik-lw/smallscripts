### This folder will generate the QR codes for the text list in 'DOIs' folder using Google APIs.
### Author: Aik with guidance from Syed Abdullah Aljunid
### Date: Oct 2019

### Software Prerequisite
### Bash with wget package
### You can run the Bash file on Cygwin.

### The file to run is in Bash_files/bash_QR.sh

### How to run
### 0. Run Cygwin
### 1. Navigate to the Bash_files folder with cd and dir Linux commands.
### 2. bash bash_QR.sh


### Variables
### Change the QR codes pixel sizes on $QRsize variable in Bash_files/bash_QR.sh
